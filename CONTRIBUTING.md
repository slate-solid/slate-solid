# Contributing to SlateSolid

## Local Dev

1. Clone this repo.
1. `npm install`
1. `npm start`

## Icons

Icons for README were generated using `SvgLetterIcon`

```jsx
<SvgLetterIcon bg="#477bbe" fg="#fff">
S
</SvgLetterIcon>
<SvgLetterIcon bg="#d8d8d8" fg="#505050">
S
</SvgLetterIcon>
```

## Npm Publishing

- Update `package.json` versions
- Run `npm i`
- Run `npm run publish:npm`
